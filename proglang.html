Chapter 6: Programming Languages
Machine language Programming Language that is converted from High Level language for the machine to use.
Debugging Removing errors from the code to get it to function properly.
Assembler Converts instructions from a high level language to Machine Language.
Assembly language Very low level language.
Translator Sort've like an Assembler. Converts Machine Language to High Level Language.
Compiler Compiles code in order to find correct syntax.
Interpreter Processes the code from the High level language and runs it.
Cross-platform software Applications that can run on multiple platforms and operating systems.
Natural language Language that humans can speak and talk with.
Formal language A language that is directly related to syntax and how you type it. I.E. A Programming Language.
Machine independence Software can run on multiple machines without having to tweak it.
High-level programming language A Programming Language that is not low level, Allows for more design and task automation that can be automated inside the language.
Object-oriented programming (OOP) Use objects to design and organize codes
Scripting language Can be used to create scripts that can simplify repetitive tasks.
Data types Different forms of data that can be represented in a programming language.
  • Integer
    • Consists of whole numbers
  • Float
    • Contain values other than whole numbers like decimal values. We would call these floating point values.
  • String
    • Sequence or pattern of characters for things like sentences or words.
Primitive data type Characters in programming language (int) or (char)
Data structure How you can group or arrange data.
Python collection data strcuture
  • List
    • Created by sqaure brackets
  • Tuple
    • Ordered and unchangeable
  • Set
    • Unordered, unchangeable, unindexed
  • Dictionary
    • Ordered, changebale, do not allow duplicates
Function Set of instructions to preform a task
Local variable Referenced only within a function
Global variable Not a restricted variable, can be called from outside of a function.
Source program Original version of software.
Object program Assembly process of source program
