Networking and Internet
Term: Definition:
Network Classification Method of classifying networks.
  • LAN
    • Local area network.
  • MAN
    • Metropolitan area network.
  • WAN
    • Wide area network.
Network models
  • Bus network
    • Each network component is connected to a single cable.
  • Star network
    • Each network component is connected to a central hub.
Protocols Beurocracy for Network Connections.
  • CSMA/CD Ethernet
    • Manage access to Ethernet networks while also mitigating potential collisions.
  • CSMA/CA Wifi
    • Manage access to Wifi networks while also mitigating potential collisions.
Combining networks: Expand the communication system to create a more extensive network.
  • Repeater
    • Facilitates communication between buses with signals exchanged back and forth.
  • Bridge
    • Establishes a connection between buses using destination addresses.
  • Switch
    • Acts as a bridge with multiple connections for enhanced network communication.
  • Router
    • Forms connections between networks to create the internet.
  • Gateway
    • The juncture where a network is linked to the internet.
Client/server model Communication between processes.
  • P2P:peer-to-peer
    • Communication between two processes.
  • Distributed system
    • Communication between multiple processes.
  • Cloud computing
    • Communication between multiple processes over the internet.
Internet Addressing Allocates addresses for computer systems.
  • Tier 1
    • Direct connection to the internet.
  • Tier 2
    • Direct connection to routers that have experienced failure.
  • Tier 3
    • Direct connection to routers that have experienced failure.
IP address Internet Address.
  • ICANN - Internet Corporation for Assigned Names and Numbers (ICANN)
  • DNS - Facilitates address translation.
  • TLD: Top-level domain - Classification for commercial institution domains.
  • Country-code top-level domains (ccTLDs)
Internet protocols
  • FTP
    • Software for catologing and duplicating files over a network.
  • HTTP
    • Manages webpages.
  • SMTP
    • The manner in which two computers communicate during the transmission of email messages.
World Wide Web
  • URL
    • Address to retrieve documents hosted on the web.
  • Client-side activities
    • Activities preformed by a client.
  • Server-side activities
    • Activities preformed by a sever.
Internet four-layer software Provides structure to application software.
  • Application layer
    • Gathers and delivers messages to designated addresses.
  • Transport layer
    • Segments messages into packets.
  • Network layer
    • Allocates addresses to individual packets.
  • Link layer
    • Conveys the packets.
Internet is:
  • Packet-switching network
    • Data is divided into packets for efficient transmission.
  • Fault tolerant
    • Refers to a system or network's ability to continue functioning without interruption.
  • Redundant
    • Provision of backup or duplicate elements to enhance reliability and prevent system failure.
Cybersecurity The practice of protecting systems, networks, and programs from digital attacks.
  • Malware
    • Any software that is harmful to a computer.
  • Virus
    • A program that replicates itself and infects a computer.
  • Worm
    • A program that replicates itself and spreads to other computers.
  • Trojan horse
    • A program that appears to be useful but actually causes harm.
  • Spyware
    • A program that monitors a user's activity without the user's knowledge.
  • Phishing
    • A technique for obtaining unauthorized data.
  • DDoS attack
    • A technique for crashing a website by overwhelming it with data.
Protection
  • Firewall
    • A system that prevents a specific type of information from moving between untrusted networks.
  • Spam filter
    • A program that blocks unwanted e-mail.
  • Proxy server
    • A server that acts as an intermediary between a client computer and the Internet.
  • Antivirus software
    • Software that is specifically designed to detect viruses and protect a computer and files from harm.
  • Cryptography
    • The science of transforming information into a secure form.
  • Public key encryption
    • An encryption system that uses two keys: a public key for everyone and a private key for the recipient.
