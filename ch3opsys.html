Operating Systems Terminology

Term: Definition:
Operating System An Operating system is the software that controls the overall operation of a computer.
Job Queue A queue that resides in mass storage where jobs wait for execution
FIFO First in First out
Interactive Processing Allows a program to be executed while carrying on dialogue with the user
Time-Sharing An operating system that provides services to multiple users at the same time
Multiprogramming The running of two or more programs simultaneously by a computer with more than one CPU
Application Software Consists of programs for performing tasks particular to the machine's utilization.
System Software Type of computer program that is designed to run a computer's hardware and application programs.
Utility Software Consists of software programs for performing activities that are fundamental to computer installations but not included in the operating system.
Graphical User Interface (GUI) Objects that are manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell Older "GUI's" communicated with users through textual messages using a keyboard and monitor screens.
Kernel Internal part of the operating system that performs the very basic functions required by the computer installations.
Device Driver Software units that communicate with the controllers
File Manager A computer program that provides a user interface to manage files and folders.
Memory Manager The process of controlling and coordinating a computer's main memory.
Virtual Memory A memory management technique where secondary memory can be used as if it were a part of the main memory
Boot Strapping Bootstrapping refers to a self starting process that is supposed to continue or grow without external input
ROM A type of non-volatile memory used in computers and other electronic devices
Deadlock A situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
Spooling The practice of holding data in temporary storage for execution by another device or program.
Super User (or administrator) A user that can make changes on a computer that will affect other users of the computer.
Multi-factor Authentication An Authentification system that requires more than one distinct authentication factor for successful authentication
